Insurance Charges Prediction
Overview

This Machine Learning project predicts medical insurance charges based on customer information such as age, gender, BMI, number of children, smoking status, and region.

Feature Scaling using StandardScaler

Linear Regression Model Training

Results

The Linear Regression model achieved an R² Score of 0.80, meaning it can explain approximately 80% of the variation in insurance charges. This demonstrates strong predictive performance and effective feature engineering.
